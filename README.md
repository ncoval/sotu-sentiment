# 2026 State of the Union — Sentiment Analysis Dashboard

Interactive visualization analyzing all 294 sentences from President Trump's 2026 SOTU address.

## Deploy to Vercel (fastest — ~2 min)

1. Push this folder to a new GitHub repo
2. Go to vercel.com → "Add New Project"
3. Import the repo → Vercel auto-detects Vite → Click Deploy
4. Done — you'll get a public URL

## Run locally

```bash
npm install
npm run dev
```
